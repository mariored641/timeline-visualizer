## תהליך מהיר:

1. **הורד/העתק את הקובץ הזה**
2. **ערוך את הרשימה בתחתית הקובץ** - כתוב אילו אנשים ו/או אירועים אתה רוצה
3. **העלה לצ'אט AI** (Claude/ChatGPT/Gemini) ואמור:
   > "צור לי קובץ JSON לפי ההנחיות בקובץ המצורף"
4. **העתק את ה-JSON שקיבלת** והדבק באפליקציה דרך: ייבוא → הדבק טקסט JSON

1. name, short_name, description → תמיד בעברית
2. id → תמיד באנגלית, אותיות קטנות, מילים מופרדות ב-_ (קו תחתון)
3. categories (אנשים) → מערך, חובה לפחות אחת. דוגמה: ["judaism", "philosophy"]

6. end_year → null אם האירוע חד-פעמי (לא טווח)

══════════════════════════════════
  הנחיות ל-short_name (חשוב!)
══════════════════════════════════

ה-short_name מוצג כשהציר זמן מוקטן. צריך להיות קצר ומזוהה מיד.

• מלחינים / מדענים / אמנים → שם משפחה: בטהובן, מוצרט, איינשטיין, פיקאסו
• מנהיגים → השם המוכר: בן גוריון, צ'רצ'יל, נפוליאון
• רבנים ופוסקים → הכינוי המקובל: הרמב״ם, רש״י, הבעש״ט, החפץ חיים
• אדמו"רים חסידיים ודמויות דתיות → תעדיף תמיד כינוי מוכר או שם משפחה. השתמש בכינוי המזוהה ביותר.
  במקרים של דמויות רבניות ניתן להשתמש בשם של ספר אם זהו הכינוי הנפוץ ביותר (למשל 'השפת אמת').
  אל תשתמש בשם החצר כברירת מחדל אלא אם הוא באמת המזהה הנפוץ ביותר.
• קדושים נוצריים → הכינוי המוכר: אוגוסטינוס, תומאס אקווינס
• אם אין כינוי מקובל → שם משפחה פשוט

** הערה: לפעמים המידע על כינוי מקובל לא נמצא בוויקידאטא.
   במקרה כזה, השתמש בידע שלך או חפש באינטרנט מהו הכינוי הנפוץ ביותר. **

תהליך עבודה חובה לכל אדם/אירוע:
1. חפש את הדמות בוויקיפדיה (עברית או אנגלית)
2. מתוך דף הויקיפדיה, העתק את ה-URL המדויק
3. מתוך דף הויקיפדיה, לחץ על הקישור לוויקידאטא (בסרגל הצד) וקח את ה-Q-ID
4. אם לא מצאת דף בוויקיפדיה → שים null בשני השדות

אם אתה לא מכיר דמות מסוימת (למשל ביקשו ממך "הרב הנזיר" ואתה לא בטוח מי זה):
→ חפש באינטרנט! אל תנחש שמות. אם חיפשת ולא מצאת → כתוב הערה בסוף.

• name → השם המלא המדויק. חפש בוויקיפדיה! אל תמציא שם אדם בשום מקרה
• wikidata_id → רק מוויקידאטא בפועל. אסור לנחש Q-ID! אם לא מצאת → null
• wikipedia_url → העתק URL ישירות מוויקיפדיה. עברית עדיף, אנגלית אם אין עברית, null אם אין בכלל
• image_url → תמיד null! האפליקציה שואבת תמונות אוטומטית
• שנות לידה/פטירה → מוויקיפדיה/וויקידאטא. אם אין → null
• description → תיאור עובדתי קצר בעברית. רק עובדות מאומתות
• short_name → כינוי מקובל. אם לא בטוח, חפש באינטרנט
• categories → סווג לפי תחום הפעילות העיקרי. אם רלוונטי ליותר מתחום אחד, הוסף כמה
• primary_location → המקום בו האדם חי/פעל רוב חייו
• secondary_location → אם היגר או עבר מקום משמעותי, ציין את המקום החדש + שנת המעבר

אם לא הצלחת לאמת דמות מסוימת, הוסף בסוף ה-JSON (אחרי הסוגריים) הערה:
// לא הצלחתי לאמת: [שם הדמות] - [הסיבה]
